# GharGhar Sewa

GharGhar Sewa is an AI-powered MERN stack home service platform developed for Nepal. The system connects users with service providers such as electricians, plumbers, cleaners, and technicians.

## Features

- JWT Authentication
- Role-Based Access Control
- AI Chatbot using Rasa
- Hybrid Recommendation System
- Booking Management
- Review and Rating System
- Provider Dashboard
- Admin Dashboard

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI / Recommendation
- Rasa Chatbot
- DIET Model
- Cosine Similarity
- Haversine Formula

## Recommendation System

The system uses:
- Context-Based Filtering
- Content-Based Filtering
- Weighted Provider Ranking

## Testing

- Jest
- Postman
